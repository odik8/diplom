# Документация проекта

Полная документация системы онлайн-заказа и доставки готовых блюд.

## Содержание

- [Отчёт о покрытии кода тестами](TEST_COVERAGE.md) — результаты модульного и
  интеграционного тестирования (120 тестов, покрытие бэкенда 82 %).
- [Скриншоты интерфейса](screenshots/) — экраны мобильного приложения и
  веб-панели администратора.

## Архитектура и API

Описание архитектуры (трёхуровневая клиент-серверная, диаграммы C4), модели данных
и полный перечень REST-эндпоинтов приведены в корневом [README.md](../README.md)
и в пояснительной записке ВКР (`VKR.docx`, главы 2–3).

## Запуск и развёртывание

Инструкции по установке, локальному запуску и развёртыванию через Docker Compose —
в разделе [«Установка и запуск»](../README.md#установка-и-запуск) корневого README.
